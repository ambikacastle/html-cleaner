Use this tool to clean up HTML that you've exported from a WIRED story in InDesign.

By default, InDesign adds a bunch of messy markup to an exported HTML file that we don't need. This tool removes all those extraneous tags, and produces an HTML file that you can use in something like WordPress. It keeps paragraph, strong, and emphasis tags.

To export an HTML file from InDesign, type command + E.
